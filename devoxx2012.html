Lambda in collection API

external iteration 
					
for (Shape s : shapes) {
    s.setColor(RED);
}
internal iteration 
shapes.forEach(s -> { s.setColor(RED); });
streams (aka pipes) 
					
int sum = shapes.stream()
                .filter(s -> s.getColor() == BLUE)
                .map(s -> s.getWeight())
                .sum();

parallel collections 
					
int sum = shapes.parallel()

                .sum();

Type annotations

Making Java annotations more general and more useful

immutability 
						
@Immutable Date date = new Date(0);
date.setTime(70);   // compile-time error
generic type arguments: 
						
Map<@NonNull String, @NonEmpty List<@Readonly Document>> files;
arrays: 
						
Document[@Readonly] docs1;   // docs1 is an unmodifiable one-dimensional array of mutable Documents

Type annotations

typecasts: 
						
myString = (@NonNull String) myObject;
type tests: 
						
boolean isNonNull = myString instanceof @NonNull String;
object creation: 
						
new @NonEmpty @Readonly List(myNonEmptyStringSet)

Date and Time API

jsr 310 (Stephen Colebourne: Joda time)
  • Discrete Timeline: consecutive instants separated by nanosecond resolution
  • Instants: specific point on a discretized timeline
  • Partials: an indication of date or time that is not sufficient to specify a specific, unique point on the timeline. For example, "June 7"
  • Durations: a length of elapsed time in nanoseconds
  • Periods: Periods.periodBuilder().years(8).months(3).build()
  • Intervals: a period of time between two instants on the timeline
